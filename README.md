# SoFly CRUD Flask Application

## Overview
The SoFly CRUD Flask Application is a simple web-based system that demonstrates how to build a full CRUD (Create, Read, Update, Delete) application using Python, Flask, and SQLite. The application allows users to submit and manage flight requests through a user-friendly interface.

## Features
- Create new flight requests
- - View stored flight requests
  - - Edit existing requests
    - - Delete requests from the database
      - - Simple web interface built with HTML and CSS
